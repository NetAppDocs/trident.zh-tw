= 支援
:hardbreaks:
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


Astra Trident是正式支援的NetApp專案。您可以使用任何標準機制聯絡NetApp、取得所需的企業級支援。

在「Container」通路上、也有一個充滿活力的公共社群、由容器使用者（包括Astra Trident開發人員）所共同參與 http://netapp.io/slack["NetApp的Slack工作"^]。這是您提出專案相關一般問題、並與志同道合的同儕討論相關主題的好地方。
